# ConveyorBeltCAD: A conveyor belt modelling exercise 

A complete CAD project with provided 3D models and assemblies, also videos of motion studies and 2D technical drawings and exploded view. Using solidworks, steps files are also included for model transfer purposes.
---

## 📂 Project Structure

```text
├── CAD models.zip                # Main compressed file containing SolidWorks parts and assemblies
│   ├── ConveyorBeltv2.SLDASM     # Top-level conveyor system assembly
│   ├── x3axGuide.SLDASM          # 3-Axis linear guide sub-assembly
│   └── [Individual Part Files]   # Frame brackets, rollers, linear bearings, and anchor blocks
├── README.md                     # Project documentation and setup guide
└── (Simscape Export Outputs)     # Generated physical XML schema and geometry (.STL) files
```
